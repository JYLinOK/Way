Main features
1. Open source based on the LGPL-2.1 License.

2. A simple and fast front-end generator library framework.

3. Way is an individual assisted library, or can be called a simple framework.

4. Modular development can be carried out quickly to avoid repetitive tedious code labor.

5. You can edit the front page quickly by means of WYSIWYG, and compatible with various frameworks at the same time.

6. You can intelligently configure routes on multiple pages at the same time to avoid duplicate routes and different configurations of link paths.

7. This framework library is Mr. Jinwei Lin's main open source product at present, and keeps excellent update speed and frequency.

8. This software allows you to use frame open speed and fun, fast development of native JS or TS and HTML website front-end.

9. The front-end page generated through the componentized compilation of this software is the native JS or TS and HTML website page, which naturally supports SEO.

10. The front-end page generated by componentization writing of this software can eliminate all processing traces. Make your web pages look as if they were written by hand.
